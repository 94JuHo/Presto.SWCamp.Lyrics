## 1. 주요 기능

1. LRC 가사 파일 출력 (특수 문자 포함)
1. 가사 위치 이동 지원
1. 다중 라인 출력 지원
1. 전체 가사 출력 지원
1. 항상 위(TopMost) 지원
1. 자석 윈도우 지원
